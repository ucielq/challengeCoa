Instrucciones para levantar el challenge:
-Ejecutar el script  "Script-Database-Sql" configurando la ruta de los FileNames a la ruta local donde tenga instalado MSSQL.
-Modificar el ConnectionString del appsettings.json con el nombre de servidor SqlServer donde tenga levantada la base de datos del punto anterior.
-Para las consultas en Postman desactivar la verificacion de certificado SSL.
-En caso de tener el puerto configurado en launchsettings.json ocupado, reemplazarlo por uno libre.
